TurnData

TurnDataの連想配列は以下のようになっています。

 
turnData : {
  turn : 現在のターン数(0-600:int)
  myId : 自分自身のプレイヤーID(0-2:int)
  playerCount : プレイヤーの人数(3:int)
  playerList : {
    [
      id : プレイヤーID (0-2:int)
      x : プレイヤーのx座標 (0.000-1000.000:float)
      y : プレイヤーのy座標 (0.000-1000.000:float)
      angle : プレイヤーの角度 (右向きから反時計回りに0-360:int)
      state : 通常状態、攻撃状態、スタン状態
      coolTime : 攻撃をしてからの待機時間 (0になると攻撃可能)
      stunTime : ダメージを受けて気絶している時間(0になると移動可能) : int
      lastAttackId : 攻撃シーンのとき、最後に攻撃したキャラクターのID
    ],/* 3個並ぶ */
  }
  coinCount : コインの数(0-:int)
  {
    [
      id : コインID (0-:int)
      type : 0が入ります。ゲームに使用しません
      x : コインのx座標 (0.000-1000.000:float)
      y : コインのy座標 (0.000-1000.000:float)
      appearTime : コインが出現するまでの時間 (0になるまでは出現せず)
      releaseX : コインが散らばり始めた場所のx座標
      releaseY : コインが散らばり始めた場所のy座標
    ], /* コインの数だけ続く */
  }
}
}
