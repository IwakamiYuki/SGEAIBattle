AIバトルドキュメント

内容

コイン集めゲーム

ルール

  • フィールドに落ちているコインを集め制限時間内により多く集めることができたAIが勝利します。
  • 相手のAIを近距離攻撃することができます。攻撃を受けたAIは所持コインの一部をフィールドに落とします。
  • 攻撃は当たり具合によって威力が違い、威力が高いほど落とすコインの枚数が変わります。
  • 最大威力で攻撃がヒットした場合、攻撃を受けた相手は一定時間行動できなくなります。

プレイヤー

座標は左下を(0,0)とし、角度は右向きx方向から反時計回りに0〜360で表されます。 ゲームは3人で行われます。

各プレイヤーの初期配置は

  1. (500,000, 166.666)
  2. (788676,666.666)
  3. (211.324.666.666)

で向きは

  1. 90
  2. 210
  3. 330 を向いています。

プレイヤーは半径80.000を持っており、フィールドの外に出たり、互いに重なることはありません。

ターン

各プレイヤーの一つの行動選択を1ターンとします。 あるプレイヤーが行動してから、そのプレイヤーが再び行動を行うまで 、3ターンが経過します。

ゲームは1800ターン(各600)ターン行われます。 再生機では、20ターンを60秒で再生します。

移動アクション

移動アクションを行います。

以下の時、移動アクションは行われません。

  • 攻撃アクションを行うターン
  • 攻撃アクションがヒットしてから16ターン
  • 強攻撃を受けてから40ターン

向きの変更

自分の行動ターン、必ずプレイヤーは向きを変化させることが出来ます。 向きは-12度〜12度の範囲で変化できます。

攻撃アクション

キャラクターが攻撃アクションを行った時、 プレイヤーキャラクターが向いている方向に、プレイヤーキャラクターの中心から 2 プレイヤー半径 (RADIUS)進めた位置と、 敵の中心点からの距離 (dist) が RADIUS 未満であれば、攻撃はヒットします。 攻撃力(ATK)は(2 RADIUS - dist) / RADIUS * 50で、 攻撃を受けたキャラクターはコインを放出します。

放出されるコイン数は ターゲット所持数 * 攻撃力 / 200 + 1枚 で コインはターゲットの中心から100000〜200000の距離で、プレイヤーが獲得可能な位置に出現します。

実行方法

java -jar arena.jar
 
の後に各言語と、その実行ファイル名を以下のような形で3つ並べて書いてください。
 
-java java/AICore.class
-cpp cpp/a.out
-js node/ai_core.js
-cs cs/ai_core.exe
-php php/ai_core.php

例えばjavaのサンプルAIを実行する場合は

java -jar arena.jar -java java/AICore.class -java java/AICore.class -java java/AICore.class

としてください。

再生プレイヤー

Unityの再生プレイヤーを用意しました。 プロジェクトファイルを開いていただいて、 リプレイデータを設定していただくことで再生できます。
